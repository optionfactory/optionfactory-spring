# optionfactory-spring/authentication-token

Authentication via HTTP headers (opaque tokens, jws, jwe) for Spring Security.

## Maven

```xml
        <dependency>
            <groupId>net.optionfactory.spring</groupId>
            <artifactId>authentication-tokens</artifactId>
        </dependency>
```
